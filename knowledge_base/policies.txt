深度求索科技有限公司 - 政策文件
================================

隐私政策
--------
最后更新：2025年1月1日

1. 数据收集
我们收集以下数据：
- 账号信息：邮箱、姓名、所属组织
- 使用数据：API 调用次数、令牌数量、时间戳
- 支付信息：由 Stripe 处理（我们不存储完整信息）
- 技术数据：IP 地址、用户代理、请求模式

我们绝不：
- 向第三方出售个人数据
- 使用客户提示/输出训练模型（除非用户主动选择加入）
- 与广告商共享数据

2. 数据保留
- API 日志：保留30天用于调试
- 账号数据：账号存续期间保留
- 已删除账号：90天内彻底清除数据
- 对话历史：保留至用户删除为止

3. 用户权利
您有权：
- 访问您的个人数据
- 导出您的对话历史
- 删除账号及所有关联数据
- 退出模型训练数据收集

服务条款
--------
1. 可接受使用
您同意不使用我们的服务从事以下活动：
- 生成违法内容
- 骚扰、仇恨言论或歧视
- 生成恶意软件、漏洞利用或有害代码
- 自动化社交媒体操控
- 任何违反适用法律的活动

2. 知识产权
- 用户保留其提示和输出的权利
- DeepSeek 保留模型权重和基础设施的权利
- 开源模型遵循各自的许可证

3. 服务可用性
- 目标正常运行时间：99.9%（SLA保障）
- 计划维护：提前48小时通知
- 紧急维护：立即通知

4. 责任限制
- 服务按"现状"提供
- DeepSeek 不对模型输出/决策承担责任
- 用户负责验证关键输出结果
- 最高赔偿责任：过去12个月支付的费用

安全政策
--------
1. 漏洞报告
安全漏洞请报告至：security@deepseek.com
响应时间：48小时内
漏洞赏金计划：https://deepseek.com/security/bounty

2. 合规认证
- SOC 2 Type II 认证
- ISO 27001 认证
- GDPR 合规（可提供欧盟数据处理协议）
- HIPAA 合规（企业客户可提供业务伙伴协议）

3. 数据加密
- 传输中：TLS 1.3
- 静态存储：AES-256
- 密钥管理：AWS KMS，自动轮换
- API 密钥：bcrypt 哈希存储

退款政策

- 企业合同：按合同条款执行
- 免费额度：不可退款、不可转让
