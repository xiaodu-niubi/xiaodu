# 常见问题解答

## 账号与计费

**问：如何创建账号？**
答：访问 https://platform.deepseek.com ，使用邮箱注册。您将收到验证邮件来激活账号。

**问：有免费套餐吗？**
答：有，新用户赠送 $5 免费额度。免费套餐允许使用 deepseek-chat 模型每小时 100 次请求。

**问：如何计费？**
答：按令牌使用量计费——输入和输出令牌都计入费用。不同模型价格不同。请查看 https://platform.deepseek.com/pricing 了解最新价格。

**问：支持哪些支付方式？**
答：支持信用卡/借记卡（Visa、Mastercard、Amex）、PayPal，企业客户还可使用银行转账。

## API 使用

**问：如何发起第一次 API 调用？**
答：获取 API 密钥后，使用 /v1/chat/completions 端点。产品手册中有 Python、JavaScript 和 curl 的代码示例。

**问：API 是否兼容 OpenAI 格式？**
答：是的，我们的 API 遵循 OpenAI API 规范。您可以修改 base_url 后直接使用现有的 OpenAI SDK。

**问：API 响应速度如何？**
答：短回复通常 1-3 秒返回。长回复以每秒约 50-80 个令牌的速度流式输出。

**问：能否微调模型？**
答：企业客户可使用微调功能。详情请联系销售团队。

## 技术问题

**问：最大上下文长度是多少？**
答：DeepSeek-chat 支持最多 128K 令牌的上下文，约合 9.6 万英文单词或 200 页文本。

**问：支持视觉识别/文件上传吗？**
答：支持文本文件（txt、pdf、代码）上传。视觉识别功能将在后续版本中推出。

**问：函数调用如何工作？**
答：使用 JSON Schema 定义工具，模型智能判断何时调用。响应中包含函数名和解析后的参数。

**问：我的数据安全吗？**
答：安全。API 数据在传输中（TLS 1.3）和静态存储中均加密。除非明确选择加入，我们不会使用客户数据训练模型。

## 故障排查

**问：收到 401 错误怎么办？**
答：请检查 API 密钥是否正确且未过期。如有需要，可在控制面板重新生成密钥。

**问：回复内容被截断怎么办？**
答：增大 max_tokens 参数。deepseek-chat 默认 4096，最高可设 8192。

**问：模型似乎忽略了我的系统提示词。**
答：将重要指令放在系统提示词末尾。对于关键约束，同时在首条用户消息中重申。
